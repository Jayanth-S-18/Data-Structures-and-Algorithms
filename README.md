## 💻 How to Run the Code
To run any of the C++ files locally on your machine, you will need a C++ compiler (like g++).

1)Clone the repository:

```Bash
git clone https://github.com/Jayanth-S-18/Data-Structures-and-Algorithms.git
```

2)Navigate to the specific directory:

```Bash
cd DSA-Practice/Arrays
```

3)Compile the C++ file:

```Bash
g++ TwoSum.cpp -o TwoSum
```

4)Run the executable:

```Bash
./TwoSum
```
